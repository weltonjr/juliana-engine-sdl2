# Aeterium — Gameplay Specification

## Vision

Aeterium is a spiritual successor to Clonk Endeavor: a 2D sandbox platformer where the
world is a destructible bitmap. The player controlls several characters who digs into the earth,
blasts ore veins, collects the fragments, refines them, and uses the materials to build
structures. And fight the against the environment, creatures, ai player, other players on the 
internet or split screen.

The engine will have a complete modding support, and the base game will come with several 
packages (base, western, jungle, underwater, pirates). And new packages can be downloaded from
modders.

The player can choose from several ready scenarios from one of the packages or start a new campaign.

---

## Core Loop

1. **Explore** — scout the terrain for ore deposits and good dig sites
2. **Dig** — carve tunnels through dirt, expose rock, reach ore veins
3. **Blast** — use explosives to shatter ore into physical fragments
4. **Collect** — pick up fragments
5. **Carry** — transport fragments back to base
6. **Refine** — deposit into a refinery building, get ingots/components
7. **Build** — construct new structures (smelters, storage, elevators, platforms)
8. **Expand** — use new structures to dig deeper, access new materials, or survive the environment

## Battle Loop

1. **Core loop** - everithing from the core loop
2. **Craft** - craft weapons, armors and vehicles (catapults, ballons, ships)
3. **Build defenses** - build walls, pits, whole castles to protect against dangers
4. **Attack** - Attack the enemy base with a army and destroy all enemy characters

## Player Character

### Movement

| Action | Input | Notes |
|---|---|---|
| Walk left / right | A / D | - |
| Jump | W | - |
| Dig down | Hold Q + S | Start digging while pressing the key |
| Dig horizontal | Hold C + moving | Start digging in a straight line |
| Carry | (automatic when near fragment) | Use a inventory slot |
| Interact | E | Use/Open context menu|

### Character States (FSM)

```
IDLE ──▶ WALK ──▶ BUILDING
  │         │
  └──▶ JUMP ◀── FALL
          │
         DIG
          │
        INTERACT
```

- **IDLE**: Standing still on ground
- **WALK**: Moving at normal speed
- **BUILDING**: Stops and show animation of building
- **JUMP**: Rising after jump input; transitions to FALL when velocity.y > 0
- **FALL**: In the air with downward velocity; coyote timer active briefly after leaving ground
- **DIG**: Actively removing terrain cells; character slows or stops
- **INTERACT**: Using a building (short animation lock)

## Ore Fragmentation

This is the gameplay differentiator. When an explosion hits an ore vein:

When an hard element receives damage, it start to grow fractures, and breaking in parts based on the 
poligon generated by the fracture, that became a rigid body.

---

## Buildings

- Are placed by the player on flat terrain
- Have a HP; take damage from explosions/fire
- Have inventory
- Can emit fields (a mana crystal emits mana field)

---

## Fields

This is a complementary but important gameplay diferentiator:

The world will have fields (using a low resolotion vector field) that simulates the flow of 
something (mana, heat, dimentional stability)
